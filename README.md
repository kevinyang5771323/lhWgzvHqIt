# Python租房数据分析系统 python747

## 项目概述

本项目是一个基于Python开发的租房数据分析系统，集成了数据爬取、用户管理、房屋信息管理、租房数据管理等功能，致力于为用户提供便捷、实时的租房信息与市场分析。

## 技术栈

- 前端：HTML/CSS/JavaScript
- 后端：Python（使用Django框架）
- 数据库：MySQL
- 数据爬取：Scrapy

## 功能模块

### 用户管理

- 用户注册
- 用户登录
- 信息修改
- 用户信息安全与规范

### 个人中心

- 个人信息查看与编辑
- 密码修改

### 房屋信息管理

- 房屋新增、修改、删除、查询
- 用户评论房屋信息

### 租房数据管理

- 租房数据录入、修改、删除、统计
- 分析租房市场动态

### 房屋资讯管理

- 房屋资讯发布、编辑、删除

### 轮播图管理

- 轮播图添加、修改、删除、排序

### 租房数据可视化分析

- 通过图表展示租房数据
- 助力用户快速了解市场趋势

## 系统角色

- 普通用户：浏览、搜索、评论、管理个人账户
- 管理员：管理房屋信息、租房数据、房屋资讯、轮播图等

## 运行环境

- Python 3.8+
- Django 3.x
- MySQL 5.7+
- Scrapy 2.x

## 部署步骤

1. 安装Python环境和依赖库
2. 配置MySQL数据库
3. 运行Django项目
4. 启动Scrapy爬虫

## 目录结构

```
project/
│
├── app/                # Django应用目录
│
├── db/                 # 数据库相关文件
│
├── media/              # 存储上传的图片等文件
│
├── static/             # 静态文件（CSS/JavaScript等）
│
├── templates/          # 前端HTML模板文件
│
└── scrapy_spider/      # Scrapy爬虫目录
```

- 实时爬取安居客网站上的房屋信息
- 强大的租房数据管理功能，便于分析市场动态
- 用户友好的个人中心与房屋信息管理
